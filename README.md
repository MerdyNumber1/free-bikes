# Free bikes (Node.js GraphQL proxy, React.js client)

## Installation:
In client folder:
```sh
$ cp .env.example .env  # set services env vars here
$ yarn install
$ yarn start (or yarn build, but you need setup nginx)
```
In api folder:
```sh
$ cp .env.example .env  # set services env vars here
$ yarn install
$ yarn build
$ yarn start
```
